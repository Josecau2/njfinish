REPLACEMENT RULES APPLIED
--------------------------

Rule 1: Chakra Components (Text, Box, Badge, Heading, etc.)
  - Use fontSize prop with token value
  - DO NOT use fontSize in style object
  
  CORRECT:   <Text fontSize="sm">Hello</Text>
  INCORRECT: <Text style={{ fontSize: "14px" }}>Hello</Text>

Rule 2: HTML Elements (div, span, button, strong, etc.)
  - Use fontSize in style object with px value
  - DO NOT use Chakra tokens in inline styles
  
  CORRECT:   <div style={{ fontSize: "14px" }}>Hello</div>
  INCORRECT: <div style={{ fontSize: "sm" }}>Hello</div>

Rule 3: Always Use String with Units
  - Always use quoted string with "px" unit
  - Never use bare numbers
  
  CORRECT:   fontSize: "12px"
  INCORRECT: fontSize: 12
